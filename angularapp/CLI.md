⚡ Common Examples
Artifact	Command	Example
Component	ng generate component <name>	ng g c home → creates home.component.ts, .html, .scss, .spec.ts
Service	ng generate service <name>	ng g s auth → creates auth.service.ts and auth.service.spec.ts
Module	ng generate module <name>	ng g m admin → creates admin.module.ts
Directive	ng generate directive <name>	ng g d highlight
Pipe	ng generate pipe <name>	ng g p capitalize
Class	ng generate class <name>	ng g class models/user → creates user.ts
Interface	ng generate interface <name>	ng g interface models/user → creates user.ts
Enum	ng generate enum <name>	ng g enum models/status → creates status.ts
